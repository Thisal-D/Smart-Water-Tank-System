# ESP32 Pin Configuration  

### **Output Pins**  
- **Water Pump Relay:** `GPIO 14`  
- **Red LED:** `GPIO 13`  
- **Green LED:** `GPIO 12`  
- **Buzzer:** `GPIO 25`  

### **Input/Output Pins**  
#### **Ultrasonic Sensor**  
- **Echo (Input):** `GPIO 27`  
- **Trigger (Output):** `GPIO 26`  

### **Input Pins**  
- **TDS Sensor:** `GPIO 35`
